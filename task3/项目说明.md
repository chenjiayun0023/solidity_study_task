# NFT 拍卖市场

项目说明:

```shell
1、使用 Hardhat2 初始化项：
npx hardhat --init

2、安装依赖：
npm install @openzeppelin/contracts
npm install @openzeppelin/contracts-upgradeable
npm install @chainlink/contracts
npm install -D hardhat-deploy
npm install -D @openzeppelin/hardhat-upgrades
npm install dotenv

3、插件配置：
require("hardhat-deploy");
require("@openzeppelin/hardhat-upgrades")
require("dotenv").config()

4、本地单元测试
npx hardhat node --no-deploy 
npx hardhat test
npx hardhat test test/auctionTest.js --network localhost (指定测试文件)

npx hardhat coverage (覆盖率)
npx hardhat coverage --testfiles "test/auctionTest.js" (指定测试文件)

测试报告地址： coverage/lcov-report/contracts/NftAuction.sol.html

5、部署合约：
npx hardhat deploy --network sepolia
```
